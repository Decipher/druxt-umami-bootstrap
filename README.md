Umami Bootstrap theme for Druxt

## Install

`$ npm install druxt-theme-umami-bootstrap`

## Usage

Add module to `nuxt.config.js`

```js
module.exports = {
  modules: [
    ...
    'druxt-theme-umami-bootstrap',
  ],

  druxt: {
    baseUrl: 'https://example.com'
  }
}
```
